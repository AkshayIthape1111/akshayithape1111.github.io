Linux Workshop

Linux is an operating system or a kernel which germinated as an idea in the mind of young and bright Linus Torvalds when he was a computer science student. He used to work on the Minix OS (proprietary software) and thought that it needed improvements and he started the linux operating system development, Today's linux lead the server and cloud market. The duration of linux workshop will be two consecutive days, with eight hours session each day in a total of sixteen hours, properly divided into theory ,practical and handon sessions.

Workshop Content for Linux

  • Day 1(Session-1) Discussion:- 1.Why is it really necessary to learn linux 2.What is an operating system 3.The point to be care on multi operating system installation 4.How operating system boot (short) 5.How GUI display on monitor (short) Practical:- 1.How to do partition for multi operating system installation 2.First operating system installation (Windows 7). 3.Second operating system installation (Ubuntu). Hands On:- 1.What are multi ways To login into Linux system 2.Accessing Command line 3.Filesystem hierarchy for linux 4.Basic Commands
  • Day 1(Session-2) Discussion:- 1.Story of worlds first operating system 2.History of Unix 3.History of Linux Practical:- 1.Who is more powerful GUI or Command line 2.How bash is working Hands On:- 1.Managing local users and groups 2.Controlling access to files with linux system permission 3.How to create own commands in linux
  • Day 2(Session-1) Discussion:- 1.What is Program and Process 2.Process Life Cycle in linux 3.Process state in linux 4.What is systemd and init process in linux 5.List of system daemons and network services Practical:- 1.How process is created in linux 2.How to see Running process from RAM Hands On:- 1.Managing linux processes 2.Controlling services and daemons 3.Analyzing and storing logs
  • Day 2(Session-2) Discussion:- 1.What is the importance of man command in linux 2.Common error in linux Practical:- 1.How Deploy Live Linux Server on Cloud (AWS) 2.How Deploy Database Server on Cloud (AWS) 3.Domain Pointing 3.How to handle security for server (Enternal and Internal Firewall) Hands On:- 1.Package installation in linux 2.Setup web server and managning in linux 3.Setup database server and managning in linux 4.Deploy Wordpress site on Local System Quiz Competition
